♻ Resetar apps · limpar dados/logs
Marque os apps e clique em Instalar marcados. Ou use por app: Reinstalar (zera tudo) e Limpar dados (apaga conversas/logs/cache sem remover). Mexe só no perfil principal — os clones (perfil de trabalho) não são afetados. O log aparece em “Log ao vivo”.
Conecte o celular para listar os apps.
👯 Clone App · clones internos
Cria/remove clones dentro do app Clone App (com.pengyou.cloneapp). Requer o Clone App e o app-alvo instalados. Cada clone novo fica vazio — o login (número + SMS) é manual. Não mexa no celular durante o processo (é automação de tela).
